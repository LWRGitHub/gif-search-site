# Gif Search Website
Use this site to seach & find the gifs your looking for!

### Technologies used
- JavaScript
- HTML
- CSS
- Handlebars
- Express
- Tenorjs
- Nodemon
- Home Brew
- Node.js
- Bootstrap
- Tenor API

### Install
```
$ npm install express --save
```
```
$ npm install nodemon -g
```
```
$ npm install express-handlebars --save
```
```
$ npm install tenorjs --save
```
```
$ npm install dotenv --save
```
